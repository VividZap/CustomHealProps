### Overview :
Funny CustomItems Plugins Work with [Exiled-Exmod 9.12.1](https://github.com/ExMod-Team/EXILED),
this plugin add Modification for items with use Customitem 
### How Use this items with commands ?:
**Ci list** : For see Items id  
**Ci give** : Just type in your RA(RemotAdmin) Ci give itemid Playerid/Player name
### How put this plugin in my server ? :
Download CustomHealprops.dll and go in your server folders 
:
- DetectedServer : **@appdata** > **Exiled** > **Plugins**
- ServerHost : **.Config** > **Exiled** > **Plugins**
